# Experience and Insights

In this project, I implemented a 3-layer FNN from scratch using NumPy to classify handwritten digits from the MNIST dataset. The network consists of two hidden layers with ReLU activations and a softmax output layer, trained using mini-batch SGD. Through three versions, I explored the effects of different training splits, network capacity, learning rates, and the use of a validation set. V1 used an 80/20 random split, achieving 97.46% test accuracy; V2 employed the official 60k/10k MNIST split, improving test accuracy to 97.89%; V3 introduced a 10k validation set, increased hidden layer size, and reduced learning rate, achieving 97.97% test accuracy.

This project reinforced key insights: careful dataset splitting, proper initialization, learning rate selection, and monitoring validation performance significantly affect test accuracy. Implementing the network from scratch also provided hands-on understanding of forward and backward propagation and gradient computation.

The FNN is one of the most fundamental models in deep learning, embodying the core idea of “layered feature representation.” While its structure is simple, it serves as a foundation for understanding network training, activation functions, gradient flow, and optimization techniques. FNNs provides a solid base for learning more complex architectures such as CNN, RNN, or Transformers. Through this project, I gained hands-on experience in the design and training principles of neural networks.

Sherry asked to implement the network using NumPy instead of frameworks like PyTorch or TensorFlow to ensure a understanding of the underlying mechanisms. With NumPy, I could manually implement forward propagation, backward propagation, gradient calculation, and parameter updates. This approach makes the training process transparent, helping us understand how each step works instead of relying on high-level library abstractions. It provided an intuitive and concrete grasp of how neural networks operate internally and how training dynamics unfold, which is invaluable for learning more advanced models later.
